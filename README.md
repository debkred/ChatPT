# ChatPT

En AI-driven personlig tränare online byggd med Next.js.

## 🌟 Funktioner

- Språkval (svenska eller engelska)
- Träningsprofil-formulär
- AI-genererat:
  - Träningsprogram vecka 1 💪
  - Kostschema 🍽️
  - Motivation och förklaringar 📚
- Betalvägg (49 kr/mån eller 499 kr engångs)

## 🚀 Teknikstack

- Next.js 14
- React 18
- CSS-moduler
- Stripe (testläge)

## 🔧 Installation

```bash
npm install
npm run dev
